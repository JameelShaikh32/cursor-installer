Installation Guide

Follow the steps below for your Ubuntu version.

1

Ubuntu 24 or Greater (AppImage Desktop Entry)

The script will detect your Ubuntu version. For Ubuntu 24+, it will:

  • Prompt for your Cursor AppImage file location
  • Make the AppImage executable
  • Create a desktop entry (shortcut) in your applications menu using the included icon
  • You can launch Cursor directly from your menu, no DEB required!
chmod +x Cursor-*.*.*-x86_64.AppImage
./Cursor-<version>-x86_64.AppImage --no-sandbox
2

Ubuntu 22 or Lower (DEB Install)

The script will automatically convert the AppImage to a DEB package and install it. Progress spinner included during conversion!
