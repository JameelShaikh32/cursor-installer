Installation Guide

Get Cursor running on your Ubuntu system in minutes

1

Download the GUI Installer

Download the GUI .deb package (recommended) or Python file

# Option A: Download .deb package (Recommended)
wget https://github.com/JameelShaikh32/cursor-installer/releases/download/v1.0.0/cursor-installer-gui_1.0.0_all.deb

# Option B: Download Python file
wget https://github.com/JameelShaikh32/cursor-installer/raw/main/cursor_installer_gui.py
2

Install the GUI Installer

Install the .deb package or make the Python file executable

# Option A: Install .deb package
sudo dpkg -i cursor-installer-gui_1.0.0_all.deb
sudo apt-get install -f

# Option B: Make Python file executable
chmod +x cursor_installer_gui.py
3

Download Cursor AppImage

Visit cursor.com and download the Linux AppImage file

# Download from https://www.cursor.com/
# File: Cursor-*-x86_64.AppImage
4

Run the GUI Installer

Launch the graphical installer and follow the prompts

# If installed via .deb package
cursor-installer-gui

# If using Python file
./cursor_installer_gui.py
5

Launch Cursor

Start Cursor from Applications menu or terminal

cursor
