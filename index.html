Installation

1

Download the Installer

Clone or download the repository to your local machine

git clone https://github.com/yourusername/cursor-installer.git
cd cursor-installer
2

Download Cursor AppImage

Visit cursor.com and download the Linux AppImage file

# Download from https://www.cursor.com/
# File: Cursor-*-x86_64.AppImage
3

Run the Installer

Make executable and run the installer script

chmod +x cursor_installer.sh
./cursor_installer.sh
4

Launch Cursor

Start Cursor from Applications menu or terminal

cursor
